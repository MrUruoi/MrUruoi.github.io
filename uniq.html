[Linux] uniq - 반복되는 라인 처리

반복(중복)되는 라인을 제거할 수 있다.

예시

1. 아래의 파일에서 중복되는 라인(b)을 제거

# test.txt 
                    a
                    b
                    b
                    c
                
$ uniq test.txt 
                    a
                    b
                    c

2. 반대로 반복되는 라인만 표시 (-d, --repeated)

$ uniq -d test.txt 
                    b

3. 주의: 연속적으로 반복되는 라인만 제거된다.

# test.txt 
                    a
                    b
                    c
                    b
                
$ uniq test.txt 
                    a
                    b
                    c
                    b

b가 중복되고 있지만, 연속적이지 않기 때문에 그대로 표시. 그래서 보통 sort와 함께 사용한다.


4. sort와 함께 사용

$ sort test.txt | uniq
                    a
                    b
                    c

"|"는 파이프라고 부른다.
간단히 한 커맨드(프로세스)의 출력을 다른 커맨드의 입력으로 넘겨줄 때 사용한다.
여기서 sort test.txt의 결과(출력)이 uniq의 입력으로 사용되었다.


그 외 아래의 옵션이 자주 사용된다.
-c, --count 반복 횟수 표시
-i, --ignore-case 대소문자 무시
